# Design System Context – Fitness Landing Page

## Overall Theme

- **Theme Type:** Modern, energetic, fitness-focused
- **Mood:** Motivational, confident, positive
- **Visual Style:** Clean, minimal, sporty with bold highlights
- **Brand Personality:** Strong, friendly, empowering

---

## Typography

### Primary Font (Headings)

**Font Style:** Sans-serif, bold, modern

**Recommended Fonts:**
- Poppins
- Montserrat
- Inter

**Usage:**
- Hero headlines
- Section titles
- Call-to-action buttons

### Secondary Font (Body Text)

**Font Style:** Sans-serif, regular, highly readable

**Recommended Fonts:**
- Inter
- Open Sans
- Roboto

**Usage:**
- Paragraph text
- Descriptions
- Navigation links

### Font Weights

- **Light (300):** Small descriptions
- **Regular (400–500):** Body text
- **Bold (700–800):** Headlines & CTAs

---

## Color Scheme

### Primary Colors

#### Pink / Magenta (Accent & Brand Color)

- **Hex Code:** `#FF0054`
- **Usage:**
  - Buttons
  - Highlights
  - Icons
  - Decorative shapes

#### Black (Primary Text)

- **Hex Code:** `#1A1A1A`
- **Usage:**
  - Headlines
  - Navigation text

### Secondary Colors

#### Dark Gray

- **Hex Code:** `#6B6B6B`
- **Usage:**
  - Body text
  - Descriptions

#### Light Gray

- **Hex Code:** `#EDEDED`
- **Usage:**
  - Borders
  - Background accents

### Background Color

#### White

- **Hex Code:** `#FFFFFF`
- **Usage:**
  - Main background
  - Keeps layout clean and spacious

## Gradients

### Primary Gradient

**Pink Gradient (CTA / Buttons)**

```css
linear-gradient(90deg, #FF0054 0%, #FF3B7A 100%)
```

**Usage:** Call-to-action buttons, highlight sections, interactive elements
